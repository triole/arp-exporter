ARP Exporter

Available Endpoints

  • json
  • metrics

Examples of supported URL Parameters

  • /json?include=192.168.0.4/30 to only include a single ip or ips from a specific cidr range
  • /json?exclude=192.168.0.6 to exclude a single ip or ips from a range
  • /metrics?include=192.168.0.0/24&exclude=192.168.0.11

    • Note that exclude overwrites include in case both parameters are specified.
    Each endpoint supports the same parameters.
